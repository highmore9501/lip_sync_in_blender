## Cherry Lip Sync ↔ MMD Shape Key 对照表

| Cherry 符号 | Cherry 口型描述            | 推荐 MMD Shape Key         | 说明                                             |
| :---------: | :------------------------- | :------------------------- | :----------------------------------------------- |
|    **X**    | 空闲/静止，闭嘴放松        | `ん`（强度弱）或 Basis     | 自然闭唇状态，不发音时的默认嘴型                 |
|    **A**    | 闭嘴，用于 B/M/P           | `ん`（强度中）             | 双唇闭合有压力，比 X 略用力                      |
|    **B**    | 微张，牙齿闭合，用于 D/K/T | `い` 或 `い１`             | 嘴横向拉开，上下齿接近咬合                       |
|    **C**    | 张嘴，用于 EH 音           | `え`                       | 半开口型，嘴型较自然张开                         |
|    **D**    | 张大嘴，用于 AH 音         | `あ` 或 `あ２`             | 口腔全开，幅度最大                               |
|    **E**    | 圆形张嘴，用于 OH 音       | `お`                       | 嘴巴收圆但不太突出                               |
|    **F**    | 噘嘴，用于 W / OO 音       | `う`                       | 典型圆唇噘起，嘴前突                             |
|    **G**    | 下唇触上牙，用于 F/V       | `う` + `口横広げ`（组合）  | 模型无专用 F/V 型，需组合模拟（或直接用 B 替代） |
|    **H**    | 舌触上牙，用于 L 音        | `え` + `舌広げ`（组合）    | 模型无专用 L 型，用开口 + 舌伸展模拟             |
|    **I**    | 咧嘴露齿，用于 EE 音       | `い２` 或 `い`（强度加大） | 比 B 更夸张的横向拉开，可配合 `口横広げ`         |
|    **J**    | 用于 CH/J/SH 音            | `い`                       | 嘴微张、齿闭合，与 B 类似即可                    |
|    **K**    | 圆嘴且牙齿闭合，用于 R 音  | `お`（强度减弱）或 `え`    | 介于 E 和 B 之间的过渡状态                       |

---

## 补充：Cherry 独有的 I / J / K 说明

相比你最初提供的 `A~H` 体系，Cherry Lip Sync 多出了三个符号：

| 符号  | 说明           | 为何需要                                                                  |
| :---: | :------------- | :------------------------------------------------------------------------ |
| **I** | 咧嘴露齿（EE） | 英语中 "EE" 音嘴型横向拉伸很明显，与普通 B 有差异，单独区分可提升动画精度 |
| **J** | CH/J/SH 音     | 这类辅音嘴型略向前突且齿间有缝隙，与 B 的纯咬合略有不同                   |
| **K** | R 音           | 英语 R 音嘴型介于圆唇和开口之间，需要单独指定                             |

---

## 实际使用建议

1. **优先保证 A~F + X**：这 7 个是最常用的核心口型，映射好后就能覆盖大部分英文发音。
2. **I / J / K 可按需使用**：如果你的动画精度要求很高，建议加入；如果追求简洁，I 可用 B 替代，J 可用 B 替代，K 可用 E 替代（参见表格中 "Alternative" 列）。
3. **组合 Blend 提升效果**：对于 G、H 这类模型没有完美对应的口型，可以创建 **Blend 组**（如 `え + 舌広げ`），在 Blender 中通过 Mix 节点或驱动来混合两个 Shape Key 实现过渡。
