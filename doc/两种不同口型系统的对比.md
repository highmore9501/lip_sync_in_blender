## Lisa ↔ Cherry 口型符号对照表

| 符号  | Lisa 口型描述                 | Cherry 口型描述            | 对比说明                                          |
| :---: | :---------------------------- | :------------------------- | :------------------------------------------------ |
| **X** | 闭唇放松，静止/停顿           | 闭唇放松，静止/停顿        | **完全一致**，两者定义相同                        |
| **A** | 闭唇有压力，用于 P/B/M        | 闭唇，用于 B/M/P           | **完全一致**，两者定义相同                        |
| **B** | 微张，牙齿咬合，用于 K/S/T/EE | 微张，牙齿闭合，用于 D/K/T | **基本一致**，Cherry 把 EE 音单独分给了 I         |
| **C** | 张嘴，用于 EH/AE              | 张嘴，用于 EH              | **完全一致**                                      |
| **D** | 张大嘴，用于 AA               | 张大嘴，用于 AH            | **完全一致**（AA 和 AH 在实际发音中嘴型几乎相同） |
| **E** | 微圆口，用于 AO/ER            | 圆口张嘴，用于 OH          | **基本一致**，Lisa 的 ER 比 Cherry 多覆盖了一个音 |
| **F** | 噘嘴，用于 UW/OW/W            | 噘嘴，用于 W/OO            | **完全一致**                                      |
| **G** | 上牙触下唇，用于 F/V          | 下唇触上牙，用于 F/V       | **完全一致**（只是描述角度不同）                  |
| **H** | 舌抵上牙后，用于 L            | 舌触上牙，用于 L           | **完全一致**                                      |
| **I** | ❌ 无此符号                   | 咧嘴露齿，用于 EE          | **Cherry 独有**，Lisa 中 EE 归入 B                |
| **J** | ❌ 无此符号                   | 用于 CH/J/SH               | **Cherry 独有**，Lisa 中此类归入 B                |
| **K** | ❌ 无此符号                   | 圆嘴且牙齿闭合，用于 R     | **Cherry 独有**，Lisa 中 R 类归入 E               |

---

## 一目了然的差异总结

| 对比维度        | Lisa（8 个符号：A~H + X）                        | Cherry（12 个符号：A~K + X）                         |
| :-------------- | :----------------------------------------------- | :--------------------------------------------------- |
| **总符号数**    | 8 个（含 X）                                     | 12 个（含 X）                                        |
| **核心共有**    | A, B, C, D, E, F, G, H, X（9 个）                | 与 Lisa 完全相同                                     |
| **Lisa 独有**   | 无                                               | —                                                    |
| **Cherry 独有** | —                                                | **I**（EE 咧嘴），**J**（CH/J/SH），**K**（R 音）    |
| **设计哲学**    | 更精简，一个嘴型覆盖多个音素，依赖动画中间帧过渡 | 更细分，为特定音素单独分配嘴型，减少对过渡动画的依赖 |

---

## 实际使用建议

- 如果你的 **角色口型动画主要靠程序自动生成**（如 Oculus LipSync、Salsa 等），Cherry 的 12 符号体系**映射更精确**，生成的口型更贴合英文发音。
- 如果你 **手动 K 帧** 或 **口型数量有限制**（比如模型只有 8 个嘴型 Shape Key），Lisa 的 8 符号体系**更友好**，配合 C→E→F 这类过渡设计也能达到不错的效果。
- Cherry 多出的 **I / J / K** 在 Lisa 中分别用 B / B / E 替代即可，不会造成明显问题。
